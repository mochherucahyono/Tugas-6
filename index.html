Seputar Dunia Web Developer di Indonesia
Website Awards & WebDevTalk 2023 Dorong Ekosistem Digital
Exabytes Indonesia menggelar ajang penghargaan dan konferensi web developer bergengsi “Indonesia Website Awards & WebDevTalk 2023”. Acara ini bertujuan untuk mendorong pengembangan website yang lebih baik, terutama bagi UMKM, serta meningkatkan kesadaran akan keamanan dan kualitas situs web. Dengan menghadirkan narasumber dari perusahaan teknologi ternama seperti Microsoft dan IBM, acara ini juga diharapkan dapat memperkuat ekosistem digital di Indonesia. ​ 		Developer Lokal untuk Pengembangan Web3
Komunitas developer blockchain Indonesia, BlockDevId, bersama Manta Network dan Universitas Pembangunan Jaya Bintaro, menyelenggarakan acara "Indonesia Hacker House" pada Februari 2025. Acara ini bertujuan untuk memberdayakan developer lokal dalam pengembangan teknologi Web3, yang menekankan desentralisasi, keamanan, dan privasi pengguna. Melalui workshop dan mentorship, diharapkan para developer dapat membangun solusi inovatif berbasis blockchain. ​ 		Microsoft Proyeksikan Komunitas Developer Indonesia Berperan Global
Presiden Microsoft Asia, Ahmed Mazhari, menyatakan bahwa komunitas pengembang perangkat lunak Indonesia akan memegang peran penting di dunia. Hal ini didukung oleh pertumbuhan signifikan jumlah developer di Indonesia, dengan lebih dari 3,1 juta pengguna GitHub, menjadikan Indonesia sebagai komunitas developer GitHub terbesar ketiga di Asia Pasifik. Microsoft juga mengumumkan investasi sebesar $1,7 miliar untuk pengembangan cloud, data center, dan AI di Indonesia.
